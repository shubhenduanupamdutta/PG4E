## JSON in Python

---

- `JSON` has been in Python for a long time.
- Python has beautiful support for JSON. It is a built in module called `json`.
- It has two main functions: `json.dumps()` and `json.loads()`, which can be used to serialize and deserialize JSON data, respectively.
- `json.dumps()` takes a Python object and converts it to a JSON string.
- `json.loads()` takes a JSON string and converts it to a Python object.
- You can also use `json.dump()` and `json.load()` to write JSON data to a file and read JSON data from a file, respectively.
